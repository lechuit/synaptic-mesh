# Operator Outcome Value Scorecard Protocol v0.25.0-alpha

Defines a disabled-by-default, manual operator-run, local-only, passive/read-only, one-shot scorecard over explicit v0.24 capture artifacts. It produces human-readable non-authoritative value signals only: ADVANCE_OBSERVATION_ONLY, HOLD_FOR_MORE_EVIDENCE, or DEGRADE_QUEUE_SIGNAL.

No policyDecision, authorization, enforcement, approval/block/allow semantics, tool execution, network/resource fetch, memory/config writes, raw persistence/output, external effects, daemon/watchers, or agent-consumed machine-readable policy decisions.
