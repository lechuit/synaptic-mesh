# Operator Outcome Value Scorecard Recommendations v0.25.2

Recommendations are human-readable non-authoritative queue-value signals only:

- ADVANCE_OBSERVATION_ONLY
- HOLD_FOR_MORE_EVIDENCE
- DEGRADE_QUEUE_SIGNAL

Hold covers insufficient sample, evidence gaps, abstain/uncertainty, or validation issues. Degrade covers high noise. None of these are approval, blocking, authorization, or enforcement.
