# Passive Observation Window Protocol v0.26.0-alpha

Defines a disabled-by-default, operator-run, one-shot local passive observation window. The window chains explicit repo-local multisource read evidence, positive utility pass, usefulness/noise scoring, controlled operator review queue, local manual outcome capture, and value scorecard.

Boundary: read-only, local-only, bounded sources/items, redaction before persist, redacted evidence packet only, human-readable report only, non-authoritative, policyDecision: null, no runtime authority.
