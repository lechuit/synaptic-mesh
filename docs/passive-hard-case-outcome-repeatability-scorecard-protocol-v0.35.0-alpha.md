# Passive Hard-Case Outcome Repeatability Scorecard — Protocol v0.35.0-alpha

Defines a local, disabled-by-default, operator-run one-shot repeatability scorecard over the pinned completed v0.34.5 hard-case outcome value artifact.

Boundary: passive/read-only, explicit artifact only, human-readable report only, non-authoritative, no memory writes, no runtime integration, no enforcement, no tool/network/resource fetch, no approval/block/allow decision, and `policyDecision: null`.

This crosses a real safe barrier after v0.34: it asks whether usefulness/noise/evidence-gap labels stay stable across repeated receiver passes before any live memory/coherence step is considered.
