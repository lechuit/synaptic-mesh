# Bounded Explicit Multisource Shadow-Read Packet v0.20.1

Evidence packet records redacted evidence only for multiple explicit repo-local file sources. It persists hashed source identifiers, per-source status, per-source record counts, redacted record previews, `policyDecision: null`, `agentConsumedOutput: false`, `rawPersisted: false`, and `unexpectedPermits: 0`. A human-readable report is generated beside JSON evidence.
