# Positive Cases v0.21.1

Positive cases exercise real v0.20-style bounded explicit multisource shadow-read evidence from explicit repo-local sources. Passing evidence has valid bounds, clean redaction, sufficient records, generated report text, and returns `PASS_TO_HUMAN_REVIEW`, `observationAccepted`, `includedInReport`, and `readyForHumanReview`.

An isolated source failure can pass only when the run explicitly sets an acceptable failure threshold.
