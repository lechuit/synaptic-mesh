# Contributing

Thanks for reviewing Synaptic Mesh. For v0.1.0-rc1, the most useful contributions are:

- citation corrections;
- fixture additions;
- adversarial counterexamples;
- wording/clarity improvements;
- threat-model gaps.

Runtime/tooling integration, canary/enforcement, production use, host config changes, or L2+ operational deployment are out of scope for this release candidate.
